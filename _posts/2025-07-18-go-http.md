## 0x01 最简单的 http client.


```go
import (
    "net/http"
    "net/http/httputil"
)

func main() {
    resp, err := http.Get("https://www.chaos.luxe/")
    if err != nil {
        panic(err)
    }
    defer resp.Body.Close()
    s, err := httputil.DumpResponse(resp, true)
    if err != nil {
        panic(err)
    }
    fmt.Printf("%s\n", s)

}
```
