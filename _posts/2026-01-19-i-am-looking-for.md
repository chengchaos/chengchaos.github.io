I'm looking for ...

<!--more-->

looking for // 寻找 (指过程)

find // 找到 (指结果)

I'm looking for keys. 

Of course you are. // 你当然 ... 废话 ...

I'm looking for suit. // 我在找西装/一套衣服

hoodie // 连帽卫衣 (有帽子 乎迪)

I'm looking for food (可以没有冠词) // something to eat.

I'm looking for Bonnie. // 找人

Hi there // 非常随便的口头语 . 不知道怎么称呼别人的时候.


I'm looking for you. // 我在找你.

channels // 海峡, 水渠, 渠道, 频道;

channels of communication // 沟通渠道

change channels // 换台

I'll get Mom // 我去找妈咪


What channel is it on? - Five. 

We are looking for Will? // 问号表示不确定是这里找吗? 

Come on in. // 进来吧.
